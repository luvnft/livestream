## 🚀 Install

To get started locally running the project:

```bash
# Clone the repository
git clone https://github.com/zpuckeridge/livestream

# Copy the .env.example and make the relevant changes
cp .env.example .env.local

# Install packages
pnpm install

# Run the server
pnpm run dev
```
